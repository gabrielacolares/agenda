Documentação Sistema Agendas
  1. Instalar Nodejs e npm
  2. Bower: npm install bower
  3. AngularJS: bower install angular
  4. Execute o comando: bower install angular-route
  5. Bootstrap: bower install bootstrap
  6. Jasmine: bower install jasmine
  7. Karma: bower install karma
  8. Grunt: bower install grunt
  9. Execute o comando: grunt init
