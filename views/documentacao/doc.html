Documentação Sistema Agenda

Recomendações

  1. Instalar Nodejs e npm
  2. Bower: npm install -g bower
  3. AngularJS: bower install angularjs#1.4.5 --save
  4. Angular route : bower install angular-route#1.4.5 --save
  5. Bootstrap: bower install bootstrap
  6. Bower: bower install bootstrap

Descrição do Sistema

Sistema Agenda é uma aplicação destinada á cadastrar contatos de pessoas. A aplicação tem as seguintes funcionalidades:

  1. Cadastrar novo contato de pessoa
  2. Editar dados do contato da pessoa
  3. Deletar pessoa
  4. Filtrar contato digitando apenas o nome da pessoa
  5. Visualizar a lista de contatos de pessoas salvos no sistema

Descrição do Sistema

Contém a seguinte estrutura:

  1. agenda
  2. agenda/js/controllers: Onde estão localizados os controllers
  3. agenda/views/:Onde estão localizadas as telas do sistema
  4. agenda/bower _components:Onde estão localizados ad dependencias instaladas pelo bower: angular,angular-route,angular-fire/bootstrap,firebase

Para o backend foi utilizado o firebase utilizando firebase e angular-fire

Obs: Não foram configurados testes unitários e mocks de dados
